# Expense Tracker App for Advanced JavaScript College Course

### [Live Demo on Vercel](https://expenses-frontend-five.vercel.app/)



---

To run it locally:
- install pnpm
- pnpm dev (for development)
- pnpm build > pnpm preview

Add to .env
```
VITE_API_URL=https://npjs-expenses-backend-eu.herokuapp.com/api or localhost:5000/api
VITE_CURRENCY_API_KEY=[your-api-key]
```

## Features

- ⚡️ [Vue 3](https://github.com/vuejs/vue-next), [Vite 2](https://github.com/vitejs/vite), [pnpm](https://pnpm.js.org/), [ESBuild](https://github.com/evanw/esbuild) - born with fastness

- 🗂 [File based routing](./src/pages)

- 📦 [Components auto importing](./src/components)

- 🎨 [UnoCSS](https://github.com/antfu/unocss) - The instant on-demand atomic CSS engine.

- 😃 Use icons from any icon sets in [Pure CSS](https://github.com/antfu/unocss/tree/main/packages/preset-icons)

- 🔥 Use the [new `<script setup>` style](https://github.com/vuejs/rfcs/pull/227)

- ✅ Use [Vitest](http://vitest.dev/) for unit and components testing

- 🦾 TypeScript, of course

- ☁️ Deploy on Netlify, zero-config
